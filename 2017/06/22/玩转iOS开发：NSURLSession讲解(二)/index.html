APP的传输安全(ATS)

从iOS 9和Mac OS 10.11开始, 苹果就想把所有网络请求都全部改为HTTPS(RFC 2818), 但由于一下子转变肯定是不太可能的, 所以就有一个名为ATS的安全特性.

在新建项目里面, 我们会发没法正常的发送HTTP请求, 这就是ATS搞的鬼, 我们需要在工程里面设置一下, 详细怎么设置去找度娘吧.

如果你想了解更多的资料, 那就去苹果的官方文档看看Information Property List Key Reference中的NSAppTransportSecurity.
